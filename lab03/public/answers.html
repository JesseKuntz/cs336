Lab 03

  • 3.1
    • Node.js is the ground floor. It is the basics or platform on which one builds applications, while Express.js is a higher-level "framework" built on top of Node.js for buidling SPAs.

    • The package.json file is for specifying settings for your app. There is a bunch of fluff, but most importantly it specifies the name of the main app file and the dependencies.

  • 3.2
    • The files are "static" because they are not being generated on the fly by the client, they are served up when the server is started.

    • It does not continue to serve up the "Hello World" message because there is already something being served at the same route: the html file.
